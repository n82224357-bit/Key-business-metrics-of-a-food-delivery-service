# Обзор ключевых метрик сервиса доставки еды
Цели проекта: 
- Отразить состояние клиентской базы в городе Саранске через дашборд с ключевыми бизнес-метриками.
- Подготовить аналитический отчёт на основе визуализаций для принятия решений.
- Посоветовавшись со старшим аналитиком, вы решили разделить проект на несколько задач:
В результате были осуществлены следующие шаги:
1. Подготовлены QL-запросы в DataLens для визуализации следующих метрик:
- DAU (от англ. daily active users) — количество активных пользователей за день.
- Conversion Rate — коэффициент конверсии.
- Средний чек — средняя сумма покупки на пользователя.
- LTV (от англ. lifetime value) — совокупная ценность клиента за период.
- Retention Rate — коэффициент удержания пользователей.
2. Созданы визуализации в DataLens
3. Подготовлена аналитическая записка с выводами
